# Reuse First Discovery - Factory Multi-Task Start Queue V0

## Ideia

Criar uma fila curta controlada para iniciar multiplas tasks no FactoryOS.

## Objetivo

Pesquisar padroes leves de fila, priorizacao e roteamento local antes de ampliar automacao.

## O que comparar

- filas locais simples;
- mecanismos de priorizacao por categoria;
- padroes de planning dry-run;
- estrategias sem scheduler persistente.

## Decisao esperada

- usar pequeno customizado;
- manter capsule para docs_only e code_small;
- bloquear ou revisar o que exigir repo completo.
