# Reuse First Discovery - First Real Project Workspace Scaffold V0

## Ideia

Preparar um workspace local controlado para o primeiro projeto real sem tocar em remoto ou automação pesada.

## O que reutilizar

- convenção de workspaces locais;
- arquivos mínimos `README.md` e `PROJECT_STATE.md`;
- guardrails de segurança e semântica de dry-run.

## Decisão esperada

- scaffold local;
- sem git init obrigatório;
- sem push, deploy ou instalação de dependências.
