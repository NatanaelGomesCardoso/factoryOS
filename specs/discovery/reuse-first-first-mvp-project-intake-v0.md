# Reuse First Discovery - First MVP Project Intake V0

## Ideia

Intake inicial para o projeto `demo-simple-web-mvp` do tipo `web`.

## Objetivo

Pesquisar apenas o suficiente para estruturar PRD, SPEC e sprints sem iniciar produto real.

## O que comparar

- templates simples de MVP web;
- estruturas de scaffold minimo;
- padroes de inicio rapido sem dependencia pesada;
- fluxos locais sem API paga.

## Decisao esperada

- usar pequeno customizado;
- manter o primeiro corte documental e tecnico leve.
