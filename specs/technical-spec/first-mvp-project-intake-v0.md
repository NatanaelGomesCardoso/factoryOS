# SPEC Tecnica - First MVP Project Intake V0

## Decisao

Criar uma camada de intake para MVP que gera artefatos documentais minimos e candidatos de tarefa, mas nao executa live.

## Fluxo

1. validar nome, kind e template;
2. gerar discovery, PRD, SPEC e sprint plan;
3. produzir candidatos de tarefa;
4. gravar report local em reports/project-intakes/.

## Guardrails

- dry-run apenas;
- sem push, deploy, API paga ou secrets;
- artifacts locais pequenos e auditaveis.
