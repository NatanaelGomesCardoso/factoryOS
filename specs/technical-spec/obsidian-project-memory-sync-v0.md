# SPEC Técnica - Obsidian Project Memory Sync V0

## Fluxo

1. validar projeto;
2. compor nota curta;
3. verificar path permitido;
4. redigir conteúdo sem segredos;
5. opcionalmente escrever a nota com `--write`.

## Segurança

- `--dry-run` por padrão;
- path fora do vault permitido é bloqueado;
- sem logs grandes;
- sem conteúdo sensível.
