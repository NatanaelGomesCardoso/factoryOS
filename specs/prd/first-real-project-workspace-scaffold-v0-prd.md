# PRD - First Real Project Workspace Scaffold V0

## Problema

O FactoryOS já sabe planejar e canariar, mas ainda falta um workspace controlado para o primeiro projeto real.

## Objetivo

Criar um scaffold local mínimo para o projeto real, com arquivos base e sem ativar fluxo de publicação.

## Escopo

- criar workspace local;
- gerar README e estado inicial;
- manter push/deploy bloqueados;
- não instalar dependências;
- não iniciar git automaticamente.

## Fora de escopo

- aplicação em remoto;
- deploy;
- automação de release;
- desenvolvimento completo do produto.
