# PRD - Obsidian Project Memory Sync V0

## Problema

A memória curta do projeto precisava de uma via segura para o Obsidian.

## Objetivo

Criar ou atualizar uma nota curta, sem segredos e com path permitido.

## Fora de escopo

- logs grandes;
- sync irrestrito;
- qualquer escrita fora do vault permitido.
