# Prompt ChatGPT — Reuse First Discovery

Use este prompt antes de criar PRD, SPEC, Sprint JSON ou prompt para Codex.

## Prompt

Quero implementar a seguinte ideia:

[DESCREVA A IDEIA AQUI]

Antes de criar qualquer arquitetura ou mandar para o Codex, execute a etapa Reuse First.

Pesquise e avalie soluções maduras existentes:
- bibliotecas;
- frameworks;
- SDKs;
- templates;
- ferramentas open source;
- projetos/repositórios validados;
- plugins ou extensões;
- padrões consolidados.

Critérios:
- maturidade;
- licença;
- manutenção;
- segurança;
- custo;
- dependência de API paga;
- simplicidade;
- compatibilidade com WSL/local;
- integração com Codex/harness;
- esforço de adaptação.

Entregue:
1. soluções encontradas;
2. o que é útil;
3. o que deve ser evitado;
4. decisão: usar pronto, adaptar, usar como referência, criar pequeno customizado ou adiar;
5. impacto na arquitetura;
6. próximo PRD/SPEC recomendado;
7. prompt Codex somente depois da decisão.
