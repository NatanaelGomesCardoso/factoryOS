# First Real Project Workspace Scaffold V0

Comando local para preparar um workspace controlado do primeiro projeto real sem iniciar o desenvolvimento completo.

## Fluxo

- cria `workspaces/projects/<slug>`;
- gera `README.md` e `PROJECT_STATE.md`;
- não instala dependências;
- não faz `git init` por padrão;
- não faz push nem deploy.

## Resultado esperado

- `dry_run=true` para planejar;
- `create_workspace=true` somente no comando explícito;
- workspace local pronto para uso controlado.
