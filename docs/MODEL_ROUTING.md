# Roteamento de Modelos

## Padrão

- Microtarefas: Python/Ollama.
- Código simples/médio: Codex low ou medium.
- Revisão final de marco: GPT-5.5 no Codex, apenas se necessário.
- ChatGPT web: GPT-5.5 Thinking para arquitetura e revisão.

## Regra

A LLM local não é fonte de verdade. Regras Python de segurança sempre vencem.
