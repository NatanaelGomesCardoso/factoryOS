# Project Panel Dashboard V0

Painel read-only para visualizar projetos detectados em `workspaces/projects`
e os últimos reports do fluxo MVP.

## Regras

- somente leitura;
- não serve diretórios inteiros;
- não expõe secrets;
- mostra workspace relativo, template, kind e status resumido;
- oferece links seguros apenas para áreas permitidas.
